Understood. The current build is not acceptable for production. The main problems are:

1. **The dock is always open and reserving/overlaying space badly.** We will change it to an **auto-hide dock** that appears only when the mouse touches the bottom edge.
2. **Picom is wrong here.** Enlightenment is already a compositor, so `picom` causes the “Another composite manager is already running” error. Enlightenment’s own documentation states it is always compositing. ([Enlightenment][1])
3. **`sxhkd` is fighting Enlightenment keybinds.** Those “could not grab key” messages mean Enlightenment already owns those key combinations. Use Enlightenment’s own keybinding GUI for `Ctrl + Alt + T`, Super launcher, etc. Enlightenment supports configuring shortcuts from its GUI settings panels. ([Enlightenment][1])
4. **`gvfs-smb` exists in Arch Extra.** If it failed, the issue is likely stale mirrors/database or a temporary sync problem. The official Arch package is still there. ([Arch Linux][2])
5. **Eww can do this, but it must be structured properly.** Eww supports image icons, sliders, tooltips, and literal dynamic widgets, but GTK CSS has limitations, so we should design within those constraints instead of pretending it is a web browser. ([Elkowar][3]) ([Elkowar][4])
6. **The bottom reveal behaviour should be driven by `xdotool`.** `xdotool getmouselocation --shell` gives mouse coordinates, which is exactly what we need for a reliable bottom-edge dock reveal script. ([Arch Manual Pages][5])

Below is the replacement layer. This overwrites the broken dock/action-centre setup.

#!/bin/bash

set -euo pipefail

MODE="${1:-status}"
MODE_DIR="/var/lib/ib-performance-mode"
MODE_FILE="$MODE_DIR/status"

mkdir -p "$MODE_DIR"

set_governor() {
    GOV="$1"
    for CPU_GOV in /sys/devices/system/cpu/cpu*/cpufreq/scaling_governor; do
        [ -w "$CPU_GOV" ] && echo "$GOV" > "$CPU_GOV" || true
    done
}

set_epp() {
    EPP="$1"
    for CPU_EPP in /sys/devices/system/cpu/cpu*/cpufreq/energy_performance_preference; do
        [ -w "$CPU_EPP" ] && echo "$EPP" > "$CPU_EPP" || true
    done
}

set_min_freq_policy() {
    POLICY="$1"

    for CPU_DIR in /sys/devices/system/cpu/cpu*/cpufreq; do
        [ -d "$CPU_DIR" ] || continue

        MIN_FILE="$CPU_DIR/scaling_min_freq"
        MAX_FILE="$CPU_DIR/scaling_max_freq"
        CPUINFO_MIN="$CPU_DIR/cpuinfo_min_freq"
        CPUINFO_MAX="$CPU_DIR/cpuinfo_max_freq"

        [ -r "$CPUINFO_MIN" ] || continue
        [ -r "$CPUINFO_MAX" ] || continue

        MIN_VAL="$(cat "$CPUINFO_MIN")"
        MAX_VAL="$(cat "$CPUINFO_MAX")"

        case "$POLICY" in
            max)
                [ -w "$MIN_FILE" ] && echo "$MAX_VAL" > "$MIN_FILE" || true
                [ -w "$MAX_FILE" ] && echo "$MAX_VAL" > "$MAX_FILE" || true
                ;;
            normal|low)
                [ -w "$MIN_FILE" ] && echo "$MIN_VAL" > "$MIN_FILE" || true
                [ -w "$MAX_FILE" ] && echo "$MAX_VAL" > "$MAX_FILE" || true
                ;;
        esac
    done
}

set_swappiness() {
    sysctl -w vm.swappiness="$1" >/dev/null 2>&1 || true
}

set_dirty_ratios() {
    sysctl -w vm.dirty_ratio="$1" >/dev/null 2>&1 || true
    sysctl -w vm.dirty_background_ratio="$2" >/dev/null 2>&1 || true
}

set_pcie_policy() {
    if [ -w /sys/module/pcie_aspm/parameters/policy ]; then
        echo "$1" > /sys/module/pcie_aspm/parameters/policy || true
    fi
}

set_brightness() {
    if command -v brightnessctl >/dev/null 2>&1 && brightnessctl -m >/dev/null 2>&1; then
        brightnessctl set "$1" >/dev/null 2>&1 || true
    fi
}

set_fans_auto() {
    for HWMON in /sys/class/hwmon/hwmon*; do
        for PWM_ENABLE in "$HWMON"/pwm*_enable; do
            [ -w "$PWM_ENABLE" ] && echo 2 > "$PWM_ENABLE" || true
        done
    done
}

set_fans_full() {
    for HWMON in /sys/class/hwmon/hwmon*; do
        for PWM_ENABLE in "$HWMON"/pwm*_enable; do
            [ -w "$PWM_ENABLE" ] && echo 1 > "$PWM_ENABLE" || true
        done

        for PWM in "$HWMON"/pwm[0-9]*; do
            [ -w "$PWM" ] && echo 255 > "$PWM" || true
        done
    done
}

set_gpu_high() {
    for GPU_PERF in /sys/class/drm/card*/device/power_dpm_force_performance_level; do
        [ -w "$GPU_PERF" ] && echo high > "$GPU_PERF" || true
    done

    if command -v nvidia-smi >/dev/null 2>&1; then
        nvidia-smi -pm 1 >/dev/null 2>&1 || true
    fi

    if command -v nvidia-settings >/dev/null 2>&1; then
        nvidia-settings -a '[gpu:0]/GPUPowerMizerMode=1' >/dev/null 2>&1 || true
    fi
}

set_gpu_auto() {
    for GPU_PERF in /sys/class/drm/card*/device/power_dpm_force_performance_level; do
        [ -w "$GPU_PERF" ] && echo auto > "$GPU_PERF" || true
    done

    if command -v nvidia-settings >/dev/null 2>&1; then
        nvidia-settings -a '[gpu:0]/GPUPowerMizerMode=0' >/dev/null 2>&1 || true
    fi
}

set_gpu_low() {
    for GPU_PERF in /sys/class/drm/card*/device/power_dpm_force_performance_level; do
        [ -w "$GPU_PERF" ] && echo low > "$GPU_PERF" || true
    done
}

case "$MODE" in
    full)
        powerprofilesctl set performance >/dev/null 2>&1 || true
        set_governor performance
        set_epp performance
        set_min_freq_policy max
        set_swappiness 1
        set_dirty_ratios 40 10

EOF

#!/bin/bash

set -euo pipefail

eww daemon >/dev/null 2>&1 || true
sleep 0.5

eww close dock 2>/dev/null || true

OPEN=0
LAST_BOTTOM=0

while true; do
    if ! command -v xdotool >/dev/null 2>&1 || ! command -v xwininfo >/dev/null 2>&1; then
        sleep 2
        continue
    fi

    eval "$(xdotool getmouselocation --shell 2>/dev/null || echo 'X=0;Y=0;SCREEN=0;WINDOW=0')"
    SCREEN_H="$(xwininfo -root 2>/dev/null | awk '/Height:/ {print $2; exit}')"

    if [ -z "${SCREEN_H:-}" ]; then
        SCREEN_H=768
    fi

    BOTTOM_TRIGGER=$((SCREEN_H - 3))
    KEEP_OPEN_ZONE=$((SCREEN_H - 110))

    if [ "$Y" -ge "$BOTTOM_TRIGGER" ]; then
        LAST_BOTTOM="$(date +%s)"
        if [ "$OPEN" -eq 0 ]; then
            eww open dock 2>/dev/null || true
            OPEN=1
        fi
    elif [ "$OPEN" -eq 1 ] && [ "$Y" -ge "$KEEP_OPEN_ZONE" ]; then
        :
    else
        NOW="$(date +%s)"
        if [ "$OPEN" -eq 1 ] && [ $((NOW - LAST_BOTTOM)) -ge 1 ]; then
            eww close dock 2>/dev/null || true
            OPEN=0
        fi
    fi

    sleep 0.08
done
EOF

## 2. Fix the terminal/keybinding issue properly

Do **not** restart `sxhkd`. It is conflicting with Enlightenment. Set shortcuts inside Enlightenment itself:

```text
Right-click desktop → Settings → Settings Panel → Input → Keys
```

Set:

```text
Ctrl + Alt + T → gnome-terminal
Super          → rofi -show drun
Super + F      → nautilus
Super + A      → ~/.config/eww/scripts/toggle_window.sh action_center
```

This is the correct path because Enlightenment is the window manager and already owns many global key grabs. That is why `sxhkd` printed repeated “could not grab key” errors.

## 3. What this fixes

The new dock now does this:

```text
Mouse away from bottom edge → dock hidden
Mouse touches bottom edge   → dock appears
Dock no longer covers the terminal permanently
Dock background is only the dock, not the whole screen
Pinned app running          → dot shown
Pinned app not running      → no dot
Non-pinned running app      → appended after pinned apps
Chrome/Chromium             → only appears if installed
Right side                  → battery/power icon, network icon, sound icon, date/time, action-centre icon
Battery hover               → battery/performance status
Network hover               → Ethernet/Wi-Fi plus down/up speed
Sound hover                 → sound percentage
Battery click               → polished Eww performance panel, not ugly Rofi
Sound/action click          → action centre with volume slider
Action centre               → no Files/Terminal buttons
```

The brightness slider is now generic. On a real laptop with a backlight device, it should work through `brightnessctl`. In VirtualBox, it may correctly report “unavailable” because the VM often does not expose a real backlight device.

## 4. Manual checks after running

Run these:

```bash
eww windows
```

You should not see the dock open all the time. It should appear only after touching the bottom edge.

Check performance mode setting:

```bash
sudo ib-performance-mode full
sudo ib-performance-mode status
```

Check network detection:

```bash
nmcli -t -f DEVICE,TYPE,STATE,CONNECTION dev status
```

Check volume:

```bash
pamixer --get-volume
```

Check brightness support:

```bash
brightnessctl -m
```

If `brightnessctl -m` fails inside VirtualBox, that confirms why the brightness slider had no visible effect there.
